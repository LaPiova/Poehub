## PoeHub Language Guide

PoeHub supports **English** and **繁體中文**.

### Switch language

Use the language menu:

```text
[p]language
```

Alias:

```text
[p]lang
```

After switching, PoeHub will render **menus and help in only that language** (no mixed bilingual output).

### Localized menus

- `[p]config` / `[p]poeconfig`: settings menu
- `[p]conv`: conversation menu
- `[p]poehubhelp`: PoeHub help

If you change language, reopen the menu so the UI text refreshes.
